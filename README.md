## OriginMaster

This is a student project, which consisted of creating a full-stack React application from scratch.

## Stack of Technologies

### Front End:
HTML, CSS, Javascript, React, React-Bootstrap, Redux

### Back End:
Node.js, Express, Passport, Sequelize

### Database:
PostgreSQL
